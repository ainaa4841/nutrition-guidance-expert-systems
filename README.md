# üçé NutriGuidEX ‚Äì Nutrition Guidance Expert System

NutriGuidEX is a simple expert system built using **HTML**, **CSS**, and **JavaScript** to provide personalized nutrition guidance based on user inputs.  
The system analyzes factors such as age, gender, BMI, activity level, and symptoms to generate dietary recommendations, detect possible nutrient deficiencies, and suggest potential nutrition-related illnesses.

---

## ‚≠ê Features

- Collects user information (age, gender, height, weight, exercise frequency, symptoms)
- Calculates BMI and determines body status (Underweight, Normal, Overweight, Obese)
- Provides personalized calorie, carbohydrate, and protein recommendations
- Detects possible vitamin deficiencies using rule-based inference
- Identifies related illnesses (e.g., Scurvy, Anaemia, Night Blindness)
- Built fully using front-end technologies (HTML, CSS, JavaScript)
- Uses **Forward Chaining** inference technique

---

## üß† How It Works

1. User enters personal and health-related inputs  
2. System converts inputs into facts  
3. Facts are matched against a predefined rule base  
4. New facts are generated using forward chaining  
5. System displays:
   - BMI and body category  
   - Daily recommended calories  
   - Suggested macronutrient intake  
   - Possible deficiencies  
   - Potential related illnesses  
   - Recommended vitamin consumption
